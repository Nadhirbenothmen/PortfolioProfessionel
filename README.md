## ðŸŽ¨ Customization

### Update Personal Information

1. **Hero Section** (`src/components/Hero.jsx`):
   - Update your name and title
   - Modify social links (GitHub, LinkedIn, Email)

2. **About Section** (`src/components/About.jsx`):
   - Replace placeholder text with your bio
   - Add your photo (replace placeholder)

3. **Skills Section** (`src/components/Skills.jsx`):
   - Customize skill categories
   - Add/remove technologies

4. **Projects** (`src/data/projects.js`):
   - Add your projects with:
     - Title, description
     - Technologies used
     - GitHub and live demo links
     - Project images

5. **Contact Section** (`src/components/Contact.jsx`):
   - Update email, phone, location
   - Customize social links
   - Add form submission logic (e.g., EmailJS, Formspree)

### Color Palette

The color palette is defined in `tailwind.config.js`:
- **Primary**: #0F7FFF (Blue)
- **Accent**: #FF8C42 (Orange)
- **Background**: #FFFFFF (White)

You can easily change these colors to match your brand.
